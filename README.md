# PlayControl - Landing Page

Landing page del proyecto **PlayControl**, desarrollada para el curso.

## Descripción del proyecto

PlayControl es una solución digital que ayuda a monitorear y prevenir la adicción a los videojuegos. Permite registrar hábitos de juego, mostrar alertas y ofrecer recursos para promover un uso equilibrado de la tecnología.

## Segmentos objetivo

- Adolescentes y jóvenes gamers.
- Padres de familia que desean acompañar el uso de videojuegos.
- Profesionales de educación/salud interesados en el equilibrio digital.

## Principales características de la solución digital

- Registro y monitoreo del tiempo de juego.
- Alertas y recordatorios.
- Panel con consejos y recursos.
- Landing page con secciones de presentación, preguntas frecuentes y descargas.
